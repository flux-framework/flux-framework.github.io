# Release Notes

flux-core version 0.85.0 - 2026-05-05
-------------------------------------

## New Features
 * add CLIPlugin to accept arbitrary R shapes/JSON on the command-line
   ([#7252](https://github.com/flux-framework/flux-core/issues/7252))
 * add support for constraining jobs to assigned resources with `sdexec`
   ([#7566](https://github.com/flux-framework/flux-core/issues/7566), [#7581](https://github.com/flux-framework/flux-core/issues/7581))
 * sdexec-mapper: scale memory limits from sdexec-properties by allocation
   ratio ([#7585](https://github.com/flux-framework/flux-core/issues/7585))
 * add device containment support to IMP exec helper ([#7568](https://github.com/flux-framework/flux-core/issues/7568), [#7584](https://github.com/flux-framework/flux-core/issues/7584))
 * scheduler: allow pool class to be overridden ([#7582](https://github.com/flux-framework/flux-core/issues/7582))
 * add `afterstart` job dependency scheme and deprecate `after` ([#7571](https://github.com/flux-framework/flux-core/issues/7571))
 * python: add logger to BrokerModule class ([#7569](https://github.com/flux-framework/flux-core/issues/7569))
 * Add support for `flux.cli.plugins` namespace for core-packaged CLI plugins
   ([#7562](https://github.com/flux-framework/flux-core/issues/7562))
 * sdexec: add support for AllowedMemoryNodes ([#7551](https://github.com/flux-framework/flux-core/issues/7551))
 * sdexec: support DeviceAllow and DevicePolicy ([#7546](https://github.com/flux-framework/flux-core/issues/7546))
 * flux: add similarity check heuristic ([#7526](https://github.com/flux-framework/flux-core/issues/7526))

 * job-manager: fix inappropriate `LOG_ERR` message ([#7579](https://github.com/flux-framework/flux-core/issues/7579))
 * librlist: fix duplicate GPU detection ([#7580](https://github.com/flux-framework/flux-core/issues/7580))
 * dependency-after: satisfy `afterany` on job cancellation ([#7570](https://github.com/flux-framework/flux-core/issues/7570))
 * resource: batch events in initial `resource.journal` responses ([#7567](https://github.com/flux-framework/flux-core/issues/7567))

 * fix `flux shutdown --gc` ([#7564](https://github.com/flux-framework/flux-core/issues/7564))
 * fix double-free on jansson `*_new()` failure ([#7555](https://github.com/flux-framework/flux-core/issues/7555))
