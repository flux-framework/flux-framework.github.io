 * add 5m shutdown timeout on system instance ([#7238](https://github.com/flux-framework/flux-core/issues/7238))
 * shell: add oom.adjust ([#7237](https://github.com/flux-framework/flux-core/issues/7237))
 * job-list: stream responses from job-list ([#7224](https://github.com/flux-framework/flux-core/issues/7224))

 * python: add functions for kvs raw writes ([#7303](https://github.com/flux-framework/flux-core/issues/7303))
