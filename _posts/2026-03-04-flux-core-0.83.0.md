# Release Notes

flux-core version 0.83.0 - 2026-02-04
-------------------------------------

## New Features
 * track job time time limit when Flux is run as a Slurm job ([#7405](https://github.com/flux-framework/flux-core/issues/7405))
 * libjob: fix `flux_job_timeleft()` under foreign resource managers ([#7411](https://github.com/flux-framework/flux-core/issues/7411))
 * flux-resource: eventlog: add `--match-context` and `--timeout` options
   ([#7410](https://github.com/flux-framework/flux-core/issues/7410))
 * modprobe: allow rc1 tasks to manipulate environment of rc2 and rc3 ([#7409](https://github.com/flux-framework/flux-core/issues/7409))
 * python: add F58 encoded jobid to `JobInfo.to_dict()` output ([#7398](https://github.com/flux-framework/flux-core/issues/7398))
 * libflux: rename/deprecate public functions based on `aux_delete()` ([#7397](https://github.com/flux-framework/flux-core/issues/7397))
 * broker: add connector-local introspection ([#7395](https://github.com/flux-framework/flux-core/issues/7395))
 * broker: strengthen support for runtime attribute updates ([#7389](https://github.com/flux-framework/flux-core/issues/7389))
 * drain nodes with "terminated before first barrier" exception (multiuser
   jobs only) ([#7387](https://github.com/flux-framework/flux-core/issues/7387))
 * enhance `flux-python` for multi-version support, add `flux-pythonX.Y`
   convenience scripts ([#7381](https://github.com/flux-framework/flux-core/issues/7381))
 * broker: restrict runtime updates on some attributes ([#7356](https://github.com/flux-framework/flux-core/issues/7356))
 * shell: send EOF to write service immediately after tasks finish ([#7363](https://github.com/flux-framework/flux-core/issues/7363))
 * shell: add `shell.finish` callback ([#7362](https://github.com/flux-framework/flux-core/issues/7362))

## Fixes
 * protect job `--progress`  bar from terminal escape codes with `--watch`
   ([#7413](https://github.com/flux-framework/flux-core/issues/7413))
 * ensure starttime and expiration appear in resource set returned from
   `flux resource R` ([#7404](https://github.com/flux-framework/flux-core/issues/7404))
 * add `FLUX_HWLOC_XMLFILE_NOT_THISSYSTEM`, fix cpu-affinity=dry-run with
   non-local topology ([#7402](https://github.com/flux-framework/flux-core/issues/7402))
 * fix assorted bugs in broker and job-manager ([#7393](https://github.com/flux-framework/flux-core/issues/7393))
 * rc1: do not run fsck in recovery mode ([#7366](https://github.com/flux-framework/flux-core/issues/7366))
 * shell: shut down rexec server in `shell.finish` callback ([#7365](https://github.com/flux-framework/flux-core/issues/7365))
 * broker: simplify internal attribute interface ([#7358](https://github.com/flux-framework/flux-core/issues/7358))
